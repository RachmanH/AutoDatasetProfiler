# AutoDataset Profiler

Aplikasi analisis dataset otomatis berbasis web. Upload file CSV/XLSX dan dapatkan:
- Eksplorasi data (EDA) lengkap
- Deteksi tipe kolom & kualitas data
- Rekomendasi ML task
- Insight berbasis LLM
- Preview preprocessing
- Generate Research PRD

## Tech Stack

- **Backend:** Python FastAPI + SQLAlchemy + SQLite
- **Frontend:** React 19 + TypeScript + Vite + Tailwind CSS 4 + Recharts
- **LLM:** OpenCode Zen API
